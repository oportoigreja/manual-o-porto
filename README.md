# Manual Igreja O Porto — GitHub Pages (Modo A + Sumário)

- Modo A: não altera palavras do DOCX e preserva formatação (negrito/itálico/sublinhado e quebras).
- O sumário é gerado automaticamente a partir de linhas detectadas como “títulos” (sem mudar aparência):
  - linhas curtas em CAIXA ALTA
  - linhas curtas centralizadas
  - linhas curtas totalmente em negrito
  - ou estilos de título do Word (quando existirem)

## Atualizar no GitHub Pages

Substitua `index.html` e faça commit.
